Class Social.Facebook.Response

Description

The Response class turns an XML response from a Facebook API call into an object. Each XML node will be an instance of Response. A psuedo example could be:

  1. <!-- XML response from Facebook API call --> 
  2. <users_getInfo_response> 
  3.   <user> 
  4.     <name>John Doe</name> 
  5.     <status> 
  6.       <message>The last status message</message> 
  7.       <time>1261168463</time> 
  8.       <status_id>210488073921</status_id> 
  9.     </status> 
  10.   </user> 
  11. </users_getInfo_response> 

  1. array fields = ({ "name", "status" });  
  2. // Fetch the current users info 
  3. Response res = facebook->get_user_info(0, fields);  
  4.   
  5. string name = res->user->name->get_value();  
  6. string message = res->user->status->message->get_value();  
  7.   
  8. // Or cast 
  9. write("Hello %s\n", (string)res->user->name);  
  10.   
  11. // Print all status nodes 
  12. foreach ((array)res->user->status, Response child)
