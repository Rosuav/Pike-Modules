Class Social.Twitter.Response

Description

The response class parses an XML tree returned from a Twitter method and turns it into a mapping.

  1. string url = "http://twitter.com/account/verify_credentials.xml";  
  2. string xml = twitter->call(url);  
  3. Twitter.Response resp = twitter->Response(xml);  
  4. write("Hello %s, you said: %s\n", resp->name, resp->status->text);

Example

  1. import Social.OAuth;  
  2.   
  3. string endpoint = "http://twitter.com/users/show.xml";  
  4.   
  5. Consumer consumer = Consumer(my_consumer_key, my_consumer_secret);  
  6. Token    token    = Token(my_access_token_key, my_access_token_secret);  
  7. Params   params   = Params(Param("user_id", 12345));  
  8. Request  request  = request(consumer, token, params, Request.GET);  
  9.   
  10. request->sign_request(Signature.HMAC_SHA1, consumer, token);  
  11. Protocols.HTTP.Query query = request->submit();  
  12.   
  13. if (query->status != 200)  
  14.   error("Bad response status: %d\n", query->status);  
  15.   
  16. werror("Data is: %s\n", query->data());
